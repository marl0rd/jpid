jpid
====

Library for Control System Simulation.

Simulation and test of Transfer function. At this time only has:
    
    - Transfer Function available:
        - First Order.
        
    - Control available:
        - PI.
        
    - Available sources:
        - Constant.
        
    - Simulation testers:
        - First Order Transfer function Simulation.
        - PI Simulation.
